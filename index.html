Installation:
A. For Windows
  1. Go to the official Python download page for Windows.
  2. Download the Python 3 release
  3. Run the executable installer. For more info refer to the link above.
  4. Download the Image Harvest Link here
  5. Go to Windows and execute the Install-Package.bat
  6. Input credentials in credentials.py
  7. Run Image Harvest
B. For Mac
  1. Using Installer
    1. Go to the official Python download page for Mac
    2. Run the installer

  2. Using Homebrew
    1. Go to your terminal
    2. Install homebrew by executing this command
      3. 1

      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)


    3. Input your password
    4. Execute this command
      5. 1

      brew install pyenv

    5. Run this command to update python
      6. 1

      pyenv install 3.9.2
